👨‍💻 Dev Note When Running in devcontainer/github codespace:

hugo version # check version
hugo build -D # build drafts and build packages for theme

hugo new  --kind post post/my-post # Create a new post
cd post/my-post/

hugo server -w # Build and run site.

Note to see your site while on github codespaces:
Open the terminal and click the ports tab, and add 1313.
